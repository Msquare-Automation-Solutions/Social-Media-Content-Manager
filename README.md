# Social Content Management App

A social media content management application.

## Getting Started

1. Copy the environment template and add your GitHub token:
   ```bash
   cp .env.example .env
   ```
   Then edit `.env` and set `GITHUB_TOKEN`.

2. Install dependencies and run (instructions to be added as the project grows).

## Environment Variables

| Variable         | Description                          |
| ---------------- | ------------------------------------ |
| `GITHUB_TOKEN`   | GitHub personal access token         |
| `GITHUB_REPO_URL`| Repository URL                       |

> **Note:** Never commit your `.env` file. It is already listed in `.gitignore`.
